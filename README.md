## Pip Requirements (this is just a temporary note, we can remove it later)
The `pip-requirements.txt` file lists all of the Python modules that are needed by our project (e.g. numpy, etc.). Any time that we decide to use a new module, it must be added to the `pip-requirements.txt` file, then each will need to run `pip install pip-requirements.txt` to update their local environment.
